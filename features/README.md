# Features

Documentation for product and technical features of AM Lightspeed VT.

| Feature | Description |
|---------|-------------|
| [Interactive Video Scanner](path-scanner.md) | Scan interactive courseware JSON by URL; enumerate user paths and report segment/path durations (shortest, longest, average, total paths, total segments). |
| [Map User Completions](completions-mapping.md) | Copy chapter-level course completions from one LightSpeed VT user to another in the same system, with preview and skip/overwrite conflict modes. |
| [Check Users in Bulk](check-users.md) | Upload a CSV or XLSX and split rows into "exists in LightSpeed VT" and "missing" buckets that download as CSV. |
